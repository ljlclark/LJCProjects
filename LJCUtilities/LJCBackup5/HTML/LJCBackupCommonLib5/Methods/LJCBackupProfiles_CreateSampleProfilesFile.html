public static void CreateSampleProfilesFile(string profilesFilespec)

Creates a sample profiles file.

Remarks

The profiles text file can contain a definition for one or more backup profiles.
Each profile contains the values needed for a specific backup operation.

A backup profile starts with the "Backup" command followed by a comma and the backup name, then followed by the profile values.
The sample backup name contains the target backup drive and target codeline name. 

  Backup, CLJCProjects
  SourceRoot, C:\Users\User\Documents\Visual Studio 2022
  SourceCodeline, LJCProjectsDev
  TargetRoot, C:\Users\User\Documents\Visual Studio 2022
  TargetCodeline, LJCProjects
  ChangesFilespec, LJCProjects\ChangesLJCProjects.txt
  SkipFoldersFilespec, LJCProjects\SkipFoldersLJCProjects.txt
  IncludeFilters, *.cs|*.sln|*.csproj|*.Designer.cs|*.resx|*.config
  IncludeFilters, Doc\*.xml|*.cmd|*.txt|*.sql
  SkipFiles, ChangesFile.txt|BackupLog.txt|MissingFolders.txt
  SkipFiles,?Build*.cmd|ClearBuild.cmd|UpdateAll.cmd

The IncludeFilters and SkipFiles values can contain multiple entries.

Values with multiple entries are combined into a single list when read by the program.
