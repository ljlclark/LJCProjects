Namespace - LJCBackupCommonLib5

Property SkipFoldersFilespec

Gets or sets the skipped folder file name.

Remarks

This is the name of a file that contains partial target folder paths for folders that are intentionally excluded.

The backup program writes a partial target folder path to the file "MissingFolders.txt" when the backup detects a missing target folder.
This is notification of a source folder that is not included in the backup.

Adding a partial folder to the SkipFolders file prevents getting a missing folders entry for a source folder you intended to exclude.
This makes it easier to see the folders you did not intend to exclude.
