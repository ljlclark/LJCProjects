Namespace - LJCBackupCommonLib5

Property Filespec

Gets or sets the content filespec.

Remarks

When the backup profiles Filespec property is set, it looks for the profiles file with the suppied name.

If the profiles file does not exist, it calls the CreateSampleProfilesFile() method to create a sample profiles file.

The profiles file is then loaded into the program with the LoadProfiles() method.

Example

C#
  using LJCBakupCommonLib5;
        
  private static void Filespec()
  {
    var profiles = new LJCBackupProfiles();
    var profilesFilespec = "BackupProfiles.txt";
          
    // Test Method
    profiles.Filespec = profilesFilespec;
          
    var result = profiles.HasItems();
          
    // result
    // true
  }
