Namespace - LJCBackupCommonLib5

Property IncludeFilters

Gets or sets the included file filters.

Remarks

A list of file patterns using wild cards separated with the "Vertical Bar" (pipe) character. For example: "*.cs|*.sln|*.csproj |*.cmd"

Files that match one of these file patterns will be included in the Code Backup.

Files will be copied to the target folder if they do not exist there or the source file has changed.

Files will be deleted from the target folder if they do not exist in the source folder. This keeps the folders synched.
However, this behavior must be managed carefully to prevent losing code.
