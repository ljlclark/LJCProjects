Namespace - LJCBackupCommonLib5

Property IncludeMissingTargetFolders

Gets or sets the include missing target folders flag.

Remarks

The standard behavior is to only copy files where the target folder already exists.

The standard behavior uses this implicit way of controlling which folders get backed up instead of having to explicitly list the source folders to be included.

When a new source folder is created, it must be copied to or created in the target codeline to be included in the backup.

Setting the IncludeMissingTargetFolders property to true overrides the standard behavior and will create missing target folders.
