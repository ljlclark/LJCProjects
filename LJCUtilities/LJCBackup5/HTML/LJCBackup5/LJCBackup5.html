A Code Backup program.

The LJC Code Backup program is not a generalized backup program but is designed specifically for backing up or promoting source code and other related text files. It is not designed for backup of binary files.

When a developer is making changes to source code, it is challenging to keep track of which files were changed when it comes time to promote or backup the code. This is the task that the LJC Code Backup program is designed to automate.
