Namespace - LJCCreateFileChangesLib5

Method Run

Syntax

C#
public void Run()

Runs the create "Changes" file process.

Remarks

Filespecs are retrieved for each IncludeFilter.

First, filespecs are retrieved from the Target codeline.
A Source filespec is created based on the target filespec but in the the source codeline.
A "Delete" command is written to the "Changes" file to delete the target file if there is no source file with the same name.
This keeps the folders synched.
However, this behavior must be managed carefully to prevent losing code.

Next, filespecs are retrieved from the Source codeline.
A Target filespec is created based on the source filespec but in the the target codeline.
A "Copy" command is written to the "Changes" file to add the source file if there is no associated target file of that name.
A "Copy" command is also written to the "Changes" file to update the target file if it is different from the source file.

The developer can modify the "Changes" file before the next step. This may include actions such as removing deletes that the developer wants to verify before the actual backup is performed.

Example

C#
  using LJCBackupCommonLib5;
  using LJCCreateFileChangesLib5;

  // Runs the create "Changes" file process.
  private static void Run()
  {
    var profilesFilespec = "BackupProfiles.txt";
    var profiles = new LJCBackupProfiles(profilesFilespec);
    var profile = profiles[0];
    // or
    profile = profiles["LocalLJCProjects"];
    // or
    profile = profiles.LJCRetrieve("LocalLJCProjects");

    if (profile != null)
    {
      var createChanges = new LJCCreateFileChanges(profile.SourcePath
        , profile.TargetPath, profile.ChangesFilespec)
      {
        IncludeFilters = profile.IncludeFilters,
        SkipFiles = profile.SkipFiles,
        IncludeMissingTargetFolders = profile.IncludeMissingTargetFolders,
      };

      if (createChanges != null)
      {
        // Test Method
        createChanges.Run();
      }
    }
  }
