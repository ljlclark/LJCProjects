Contact Visit Detail Dialog

The Contact Visit Detail dialog allows the user to create and edit contact visit entries.

Form Controls

Contact Name Button

The Contact Name Button displays the Contact Person List to allow for the selection of the Contact data.

Contact Index

The Contact Index is the unique index for this Contact.

Contact Recall Button

The Contact Recall Button recalls the existing Customer information for the entered Contact Index.

Register Time

The Register Time values are the Register Date and Register Time.

If this is a new Visit Record, the Register Date is automatically populated with the current date and the Register Time is automatically populated with the current time.

Register Date Button

The Register Date Button displays the Calendar Dialog for selecting the Register Date and Time.

Enter Time

The Enter Time values are the Enter Date and Enter Time.

Enter Date Button

The Enter Date Button displays the Calendar Dialog for selecting the Enter Date and Time.

Exit Time

The Exit Time values are the Exit Date and Exit Time.

Exit Date Button

The Exit Date Button displays the Calendar Dialog for selecting the Exit Date and Time.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a empty place on the form.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired list item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

Help

The 'Help' menu item displays the help page for the current dialog.
