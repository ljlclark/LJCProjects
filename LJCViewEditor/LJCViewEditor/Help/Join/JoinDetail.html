Join Detail Dialog

The Join Detail dialog allows the user to create and edit Join entries.

Form Controls

View Name

The parent View Name. This value is read-only.

Table Name

The Join table name. This value does not allow blanks.

The Combobox contains the table names from the Data Source. The table name can be selected from the Combobox or a valid name can be entered.

Join Type

The Join type.

The user can enter any valid join type name. The most commonly used value is "Left".

Alias

The Join alias.

If this value is entered, it is used to prepend the Join Columns in the SQL statement instead of prepending with the Join Table name.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
