Unit Category Dropdown List

A Unit Category seperates the Unit Measures into separate categories such as "Weight", "Volume" and "Length".

Context Menu

The Context Menu can be displayed by clicking the right mouse button on the desired item. The menu items apply to the selected item or list.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

New

The 'New' menu item displays the Unit Category Detail dialog to allow for the creation of a new Unit System entry.

Edit

The 'Edit' menu item displays the Unit Category Detail dialog to allow for editing the selected list item.

Delete

The 'Delete' menu item deletes the selected list item.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportFiles\UnitCategory.doc'. It attempts to display the data with the application that is associated with the '.doc' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportFiles\UnitCategory.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.

Exit

The 'Exit' menu item closes the window and exits the application.

Help

The 'Help' menu item displays the help page for the current list.
