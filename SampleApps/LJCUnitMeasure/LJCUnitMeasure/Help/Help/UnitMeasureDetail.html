Unit Measure Detail Dialog

The Unit Measure Detail dialog allows the user to create and edit Unit Measure entries.

Form Controls

Unit System

This is the UnitSystem to which the displayed UnitMeasures belong. Only UnitMeasures that belong to this UnitSystem will appear in the UnitMeasure list.

Unit Category

This is the UnitCategory to which the displayed UnitMeasures belong. Only UnitMeasures that belong to this UnitCategory will appear in the UnitMeasure list.

Unit Type

This is the UnitType to which the displayed UnitMeasures belong. Only UnitMeasures that belong to this UnitType will appear in the UnitMeasure list.

Code

This is the externally defined unit abbreviation. It is appended to the unit value to indicates the unit of measurement.

Name

This is the externally defined unit name.

Alternate Name

This is the alternate unit name.

Sequence

This is the UnitMeasure sequence value. It determines the order in which the UnitMeasure appears in the list.

Description

This is the UnitMeasure description.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a empty place on the form.

If you are using a Tablet or a laptop in Tablet Mode, touch the form and leave your finger there a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

Help

The 'Help' menu item displays the help page for the current list.
