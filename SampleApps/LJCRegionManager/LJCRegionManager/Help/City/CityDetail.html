City Detail Dialog

The City Detail dialog allows the user to create and edit city entries.

Form Controls

Province

The read-only control that shows the parent province description.

Name

The unique name text.

Description

The description text.

City

The CheckBox that indicates this is a city.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
