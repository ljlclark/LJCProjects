Province Detail Dialog

The Province Detail dialog allows the user to create and edit province entries.

Form Controls

Region

The read-only control that shows the parent region description.

Abbreviation

The abbreviation text.

Name

The unique name text.

Description

The description text.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
