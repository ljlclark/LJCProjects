City Section Detail Dialog

The City Section Detail dialog allows the user to create and edit city section entries.

Form Controls

City

The read-only control that shows the parent city description.

Name

The unique name text.

Description

The description text.

Zone Type

The zone type text.

Contact

The contact name.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
