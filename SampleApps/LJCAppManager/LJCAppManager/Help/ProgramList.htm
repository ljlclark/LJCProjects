Program List

The program list contains the defined programs. Programs can be added, edited and deleted from this list.

Context Menu

New

The 'New' menu item displays the Program Detail dialog to allow for the creation of a new program entry.

Edit

The 'Edit' menu item displays the Program Detail dialog to allow for editing the selected list item.

Delete

The 'Delete' menu item deletes the selected list item.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Select

The 'Select' menu item returns the selected item to the associated parent list.

Close

The 'Close' menu item closes the list window without returning the selected item.
