User Detail Dialog

The User Detail dialog allows the user to create and edit user entries.

Form Controls

User Name

This is the name of the system user.

User ID

This is the windows user ID. It must be qualified with the machine/domain name.

Example:

DESKTOP-PDPBE34\\Les

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
