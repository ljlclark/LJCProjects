Authorized Module List

The authorized module list contains the authorized modules for the selected program. Modules can be added, edited and removed from this list.

Context Menu

Add

The 'Add' menu item displays the Module List window to allow for selecting a new module entry.

Edit

The 'Edit' menu item displays the Module Detail dialog to allow for editing the selected list item.

Remove

The 'Remove' menu item removes the selected module from the list.
Note that the module definition is not removed from the database. This can be done from the Module List window.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportAuthModule.doc'. It attempts to display the data with the application that is associated with the '.doc' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportAuthModule.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.

Close

The 'Close' menu item removes the module tabs from the Tab Display window pane.
