Program Menu Tree

The Main window left window pane is a tree view that displays the authorized programs and the associated program modules.
The authorized programs and modules are configured individually for each authorized user.

External programs and modules can be added to the App Manager program without having to modify the App Manager code. New or existing programs and modules can be added as long as they conform to the App Manager Plugin architecture requirements.

Context Menu

Show Module

This menu item is only available if a program module is selected in the menu tree.
The Show Module menu item displays the tabs for the selected module in the Tab Display window pane.

Exit

The Exit menu item exits the App Manager application.
