Program Detail Dialog

The Program Detail dialog allows the user to create and edit program entries.

Form Controls

File Name

This is the program file name. It is the name of the program file that conforms to the App Manager Plugin architecture.

Example:

AppProgram.exe

Title

This is the value that appears in the Program Menu Tree.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
