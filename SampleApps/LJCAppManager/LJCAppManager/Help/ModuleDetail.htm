Module Detail Dialog

The Module Detail dialog allows the user to create and edit module entries.

Form Controls

App Program

This is a read-only field that shows the parent Program name.

Module Name

This is the program module name. It is a type identifier that must be qualified with namespace.

Example:

LJC.AppProgram.UserModule

Title

This is the value that appears in the Program Menu Tree.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
