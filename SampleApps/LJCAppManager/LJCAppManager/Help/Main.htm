App Manager Main Window

The App Manager main window consists of two window panes separated by a moveable splitter bar.

The left windowpane is the Program Menu Tree that displays the authorized programs and program modules for the signed in user.

The right windowpane is the Tab Display pane.
When a program module is selected from the menu tree, the associated module tabs are displayed in this area.
