Facility Detail Dialog

The Facility Detail dialog allows the user to create and edit facility entries.

Form Controls

Code

This is the user-defined code for this facility. A code is an abbreviation that is easier to use than the long item description.
So if you define a facility code type for School facilities with the code FACSCH and you have a main school and two branches, you might give the schools the codes: SCHMAIN, SCHBR1 and SCHBR2. For these schools you would select the Facility Type of FACSCH.

If you have branches in multiple cities, you might include the city abbreviation in the codes. E.G. SCHCTYMAIN, SCHCTYBR1 and SCHCTYBR2.

Description

This is the facility description.

Facility Type

This is the facility type selection.
Only code types that belong to the Facility code type class will appear in this list.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a empty place on the form.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired list item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

Help

The 'Help' menu item displays the help page for the current dialog.
