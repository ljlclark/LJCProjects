Visit Filter Detail Dialog

The Visit Filter Detail dialog allows the user to enter values that are used to filter the Contact Visit List.

To 'Filter' a list means to limit the list to items that match user selected values.

The Filter values are applied with "AND" logic. This means that the items that appear in the list must match each value "and" the other entered values. So, if you put in a "Last Name" and a "First Name" the list will include only those items that match both the "Last Name" and "First Name".

The text filters match those items that begin with the entered filter letters. The matching is not case sensitive. So, if the "Last Name" filter value is "C", then it will match with any entry where the Last Name starts with a "C" or "c".

Form Controls

Begin Date

The Begin Date filter value will limit the Contact Visit List to those visits that were registered on the Begin Date.

Begin Date Button

The Begin Date Button displays the Calendar Dialog for selecting the Begin Date.

End Date

The End Date filter value will limit the Contact Visit List to those visits that were registered between the Begin date and End Date. The Begin Date and End Date are included with all the register dates between.

End Date Button

The End Date Button displays the Calendar Dialog for selecting the End Date.

Last Name

The "Last Name" filter value.

Middle Name

The "Middle Name" filter value.

First Name

The "First Name" filter value.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a empty place on the form.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired list item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

Help

The 'Help' menu item displays the help page for the current dialog.
