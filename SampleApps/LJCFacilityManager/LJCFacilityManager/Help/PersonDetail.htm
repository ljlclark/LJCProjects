Person Detail Dialog

The Person Detail dialog allows the user to create and edit individual entries.

Form Controls

Name

This is the person's full name.

First Name

This is the persons first name.

Middle Initial

This is the person's middle initial.

Principle Addressee

This indicates that correspondence should be addressed to this person.

Last Name

This is the person's last name.

Sex

This is the person's sex selection.

Person Type

This is the person type selection.
Only code types that belong to the Person code type class will appear in this list.

Address To

This is the address salutation.

Birthday

This is the person's birthday. You can type the date into the date mask field or select the button to the right of the field to display the date selection dialog.

Phone

This is the person's phone number.

Extension

This is the phone extension

Cell Phone

This is the person's cell phone number.

Fax

This is the person's fax number.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
