Unit Detail Dialog

The Unit Detail dialog allows the user to create and edit unit entries.

Form Controls

Code

This is the user-defined code for this unit. A code is an abbreviation that is easier to use than the long item description.
So if you define a unit code type for Classrooms with the code UNITCLS and you have three Classrooms, you might give the classrooms the codes: CSLA, CLSB and CLSC. For these units you would select the Unit Type of UNITCLS.

Description

This is the unit description.

Unit Type

This is the unit type selection.
Only code types that belong to the Unit code type class will appear in this list.

Beds

This is the number of beds in this unit. This may not apply for some facility units.

Baths

This is the number of baths in this unit. This may not apply for some facility units.

Phone

This is the unit phone number.

Extension

This is the unit extension.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
