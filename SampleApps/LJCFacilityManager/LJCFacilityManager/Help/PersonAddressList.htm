Person Address List

The person address list contains the assigned addresses for the selected person.

Context Menu

Add

The 'Add' menu item displays the Address List window to allow for selecting a new address entry.

Edit

The 'Edit' menu item displays the Address Detail dialog to allow for editing the selected list item.

Remove

The 'Remove' menu item removes the selected address from the list.
Note that the address definition is not removed from the database. This can be done from the Address List window.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportPersonAddress.doc'. It attempts to display the data with the application that is associated with the '.doc' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportPersonAddress.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.
