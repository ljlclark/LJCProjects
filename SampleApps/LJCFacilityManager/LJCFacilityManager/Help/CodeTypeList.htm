Code Type List

The Code Type list contains the defined code types. Code types can be added, edited and deleted from this list.

Context Menu

New

The 'New' menu item displays the Code Type Detail dialog to allow for the creation of a new code type entry.

Edit

The 'Edit' menu item displays the Code Type Detail dialog to allow for editing the selected list item.

Delete

The 'Delete' menu item deletes the selected list item.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportCodeType.doc'. It attempts to display the data with the application that is associated with the '.doc' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportCodeType.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.

Close

The 'Close' menu item removes the module tabs from the Tab Display windowpane.
