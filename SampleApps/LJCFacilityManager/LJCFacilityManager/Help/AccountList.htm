Account List

The Account list contains the defined accounts. Accounts can be added, edited and deleted from this list.

Context Menu

New

The 'New' menu item displays the Account Detail dialog to allow for the creation of a new account entry.

Edit

The 'Edit' menu item displays the Account Detail dialog to allow for editing the selected list item.

Delete

The 'Delete' menu item deletes the selected list item.

Refresh

The 'Refresh' menu item reloads the list items from the database.
