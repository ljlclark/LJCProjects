Relation Detail Dialog

The Relation Detail dialog allows the user to create and edit relation entries.

Form Controls

Person

This is a read-only field that shows the person for whom the relationship is defined.

Relation

This is the selected related person. The person seletion list is displayed by selecting the button with the ellipses to the right of the read-only text field.

Relation

This is selected relationship type.
Only code types that belong to the Relation code type class will appear in this list.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
