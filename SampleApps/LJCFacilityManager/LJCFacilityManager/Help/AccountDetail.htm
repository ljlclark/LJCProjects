Account Detail Dialog

The Account Detail dialog allows the user to create and edit account entries.

Form Controls

Person

This is a read-only field that shows the person who ownes this account.

Business

This is the selected account business. The business selection list is displayed by selecting the button with the elipses to the right of the read-only text field.

Description

This is the account description.

ID Number

This is the account ID number.

Group Number

This is the account group number.

Plan Number

This is the account plan number.

Effective Date

This is the date that the account became active. You can type the date into the date mask field or select the button to the right of the field to display the date selection dialog.

Expiration Date

This is the date after which the account is no longer active. You can type the date into the date mask field or select the button to the right of the field to display the date selection dialog.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
