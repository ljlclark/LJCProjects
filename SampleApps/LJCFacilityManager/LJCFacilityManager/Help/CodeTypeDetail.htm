Code Type Detail Dialog

The Code Type Detail dialog allows the user to create and edit code type entries.

Form Controls

Code

This is the code value. A code is a user defined abbreviation that is easier to use than the long item description.

One possible format for creating a code type value is to use the type class abbreviation followed by an abbreviation that makes it unique.

So if we are creating Person code types they might follow this pattern.
PERPAR - A person who is a parent.
PERSTU - A person who is a student.
PERCUST - A person who is a customer.

Description

This is the code type description.

Code Type Class

This is the selected code type class.
A code type class controls the code types can be selected for specific items.
E.G. A person can only be assigned a code type that belongs to the person type class.

A code type can be defined as belonging to one of 8 code type classes. A suggested abbreviation is included for each type class.
Facility Type Class - FAC
Unit Type Class - UNIT
Fixture Type Class - FIX
Equipment Type Class - EQ
Person Type Class - PER
Business Type Class - BUS
Address Type Class - ADDR
Relation Type Class - REL

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
