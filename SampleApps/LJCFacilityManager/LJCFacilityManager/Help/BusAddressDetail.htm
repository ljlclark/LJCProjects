Address Detail Dialog

The Address Detail dialog allows the user to create and edit address entries.

Form Controls

Addressee

This is a read-only field that shows the item to which this address is assigned.

Province

This is the Province/State selection.

City

This is the City selection.

Barangay

This is the Barangay selection.

Street

This is the address number and street name.

Postal Code

This is the Zip/Postal Code.

Address Type

This is the address type selection.
Only code types that belong to the Address code type class will appear in this list.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
