Business Contact List

The business contact list contains the assigned contacts for the selected business.

Context Menu

Add

The 'Add' menu item displays the Person List window to allow for selecting a new contact entry.

Edit

The 'Edit' menu item displays the Person Detail dialog to allow for editing the selected list item.

Remove

The 'Remove' menu item removes the selected contact from the list.
Note that the individual definition is not removed from the database. This can be done from the Person List window.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportContact.doc'. It attempts to display the data with the application that is associated with the '.doc' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportContact.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.
