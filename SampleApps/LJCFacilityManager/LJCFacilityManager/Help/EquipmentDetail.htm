Equipment Detail Dialog

The Equipment Detail dialog allows the user to create and edit equipment entries.

Form Controls

Code

This is the user-defined code for this equipment. A code is an abbreviation that is easier to use than the long item description.
So if you define a code type for Audiovisual equipment with the code EQAV and you have three AV units, you might give the AV units the codes: AV1, AV2 and AV3. For these AV units you would select the Equipment Type of EQAV.

Description

This is the code type description.

Equipment Type

This is the equipment type selection.
Only code types that belong to the Equipment code type class will appear in this list.

Facility

This is the Facility where the equipment is currently located.

Unit

This is the Unit where the equipment is currently located.

Make

This is the make of the equipment.

Model

This is the model of the equipment.

Serial Number

This is the equipment serial number.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
