Person Unit Detail Dialog

The Person Unit Detail dialog allows the user to create and edit person unit entries.

Form Controls

Person

This is a read-only field that shows the person assigned to this unit.

Facility

This is the Facility where the person is currently assigned.

Unit

This is the Unit where the person is currently assigned.

Begin Date

This is the date that the person is assigned to this unit. You can type the date into the date mask field or select the button to the right of the field to display the date selection dialog.

End Date

This is the date after which the person is no longer assigned to this unit. You can type the date into the date mask field or select the button to the right of the field to display the date selection dialog.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
