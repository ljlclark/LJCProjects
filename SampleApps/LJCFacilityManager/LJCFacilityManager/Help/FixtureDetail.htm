Fixture Detail Dialog

The Fixture Detail dialog allows the user to create and edit fixture entries.

Form Controls

Code

This is the user-defined code for this fixture. A code is an abbreviation that is easier to use than the long item description.
So if you define a fixture code for Air Conditioners with the code FIXAC and you have two Air Conditioner units, you might give them the codes: AC1 and AC2. For these fixtures you would select the Fixture Type of FIXAC.

Description

This is the fixture description.

Fixture Type

This is the fixture type selection.
Only fixture types that belong to the Fixture code type class will appear in this list.

Make

This is the make of the fixture.

Model

This is the model of the fixture.

Serial Number

This is the fixture serial number.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
