LJCGenDoc2 Readme

LJCGenDoc2

The GenDoc2 application generates HTML documentation from the source files. It includes external XML Comment files which are referenced by source file "include" XML comments.

The LJCGenDoc2 project group has three main projects:

LJCGenDoc2 The GenDoc2 Console Program
LJCDocDataGen The DocDataGen Library
LJCDocDataDAL The GenDoc2 Data Access Layer Library
