Template Text

The GenText Editor contains the Template Tab with the Template TextBox, Template Button and Template Text.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on the Template text. The menu items apply to the selected template file.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired list item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

Create Sections

The 'Create Sections' menu item creates sample data based on the data defined in the Template.

Generate Output

The 'Generate Output' menu item generates the Output Text based on the selected Template and Data. It performs the same function as the 'Generate Output' menu items on the Section List, Item List, Replacement List and Output Text.

Save

The 'Save' menu item saves the template using the name specified in the Template TextBox.

Exit

The 'Exit' menu item closes the window and exits the application.

Help

The 'Help' menu item displays the context sensitive Help Page.

About

The 'About' menu item displays the application About Box.
