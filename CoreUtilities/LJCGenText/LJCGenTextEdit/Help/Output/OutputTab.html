The Output Tab

The GenText Editor contains the Output Tab with the Output TextBox, Output Button and Output Text.

Output Tab Controls

Output TextBox

The Output TextBox contains the name of the file retrieved with the Output Button. It is the name of the file where the current output code will be saved if the "Save" menu item is selected from the Output Text menu.

The default output file path is:
C:\Users\UserName\Documents\Visual Studio 2022\LJCProjectsDev\CoreUtilities\LJCGenText\LJCGenTableCode\bin\Debug\DataFiles

The target saved file name can be changed by editing the name in the Output TextBox.

Output Button

The Output Button displays a File Open dialog to open the desired Output file.
