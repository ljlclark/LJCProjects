Calculation Pad

The CalcPad page is intended to be used for entering a set of simple calculations.

These calculations can be copied and pasted to a local text file. They can be copied from the text file and pasted back into the page later.

CalcPad accepts only four types of line statements. These statement types and the required order are:

Data Item Declaration
Conversion Formula
Assignment
Output Value
