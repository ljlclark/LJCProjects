Assignment

An Assignment is used to assign a value to a Data item. It has the following format: 

  DataItemName = #
  or
  DataItemName = DataItemName/Number Operation DataItemName/Number

The second and fourth positions can be a DataItem name or a number.

The operation can be one of the following: "+", "-", "*" or "/".

The value is assigned to the left side Data Item.

The following assignment sets the value for Family|Families and all related Data Items based on the Conversion Formulas.

Families = 100

The following assignment calclulaes the required water taps at one for each 125 Persons.

Taps = Persons / 125
