Value Statement

An Output Value is used to display the current value of a Data Item. It has the following format:

Value: DataItemName Rounding

The Value lines will contain the current Data Item value after the "Do Calculations" menu item is selected from the "Options" menu.

Value: DataItemName Rounding (CurrentValue)

The Rounding value is optional and can be:

Ceiling - Rounds up to the next whole (integer) value.

Floor - Rounds down to the whole (integer) value.

Round - If .5 or higher, rounds up to the next integer else rounds down to the integer value..

Round:number - Rounds to the provided number of decimal places.

The following statements will show the results in the current values.

Rapid Needs Assesment Calculation Template
  // DataItem Definitions
  Person|Persons
  Family|Families
  Community|Communities
  Village|Villages
  Tent|Tents
  LitersOfWaterPerDay
  Tap|Taps
  WaterCollectionContainer|WaterCollectionContainers
  WaterStorageContainer|WaterStorageContainers
  Toilet|Toilets
  RefuseContainer|RefuseContainers

  // Conversion Formulas
  Family is 5 Persons
  Community is 16 Families
  Village is 4 Communities

  // Initialize Values
  Families = 100

  // Calculate Needs
  Tents = Persons / 6
  LitersOfWaterPerDay = 15 * Persons
  Taps = Persons / 125
  WaterCollectionContainers = Families * 2
  WaterStorageContainers = Families
  Toilets = Persons / 20
  RefuseContainers = Families / 10

  // Show Calculated Needs
  Value: Persons Ceiling
  Value: Families Ceiling
  Value: Communities Ceiling
  Value: Villages Ceiling
  Value: Tents Ceiling
  Value: LitersOfWaterPerDay
  Value: Taps Ceiling
  Value: WaterCollectionContainers
  Value: WaterStorageContainers
  Value: Toilets Ceiling
  Value: RefuseContainers Ceiling
