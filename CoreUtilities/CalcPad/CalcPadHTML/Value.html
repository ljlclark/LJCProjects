Output Value

An Output Value is used to display the current value of a Data Item. It has the following format:

Value: DataItemName

The Value lines will contain the current Data Item value after the "Do Calculations" menu item is selected from the "Options" menu.

Value: DataItemName Rounding (CurrentValue)

The Rounding value is optional and can be:

Ceiling - Rounds up to the next whole (integer) value.

Floor - Rounds down to the whole (integer) value.

Round - If .5 or higher, rounds up to the next integer else rounds down to the integer value..

The following statements will show the results in the current values. 

    Person|Persons
    Family|Families
    Community|Communities
    Village|Villages
    Tap|Taps

    Family is 5 Persons
    Community is 16 Families
    Village is 4 Communities

    Families = 100
    Taps = Persons / 125

    Value: Persons Ceiling
    Value: Families Ceiling
    Value: Communities Ceiling
    Value: Villages Ceiling
