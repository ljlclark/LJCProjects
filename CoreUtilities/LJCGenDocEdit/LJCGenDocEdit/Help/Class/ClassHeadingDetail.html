Class Heading Detail Dialog

The Class Heading Detail dialog allows the user to create and edit Class Group entries.

Form Controls

Name

The unique item identifier.

Heading

The item heading text.

Sequence

The item sequence.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Save and New

Saves the current item and clears the detail dialog ready for a new item.

Save and Previous

Saves the current item and loads the detail dialog with the data for the previous item in the parent list ready for editing.

The dialog is closed if the current item was the first item in the list.

Save and Next

Saves the current item and loads the detail dialog with the data for the next item in the parent list ready for editing.

The dialog is closed if the current item was the last item in the list.

Help

Displays the context sensitive Help Page.
