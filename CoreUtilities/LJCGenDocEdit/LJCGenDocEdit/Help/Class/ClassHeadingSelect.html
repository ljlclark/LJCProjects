Class Heading Select List

The Class Heading Items can be added, edited and deleted from this list. A heading can be selected from this list to be assigned to a class group.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on the desired list item. The list item is selected and the menu items apply to the selected item or list.

If you are using a Tablet or a laptop in Tablet Mode, touch the desired list item and leave your finger on the item for a short while until a box is displayed. This shows that the touch is considered a right mouse click. Remove your finger and the context menu will be displayed.

New

The 'New' menu item displays the Class Heading Detail dialog to allow for the creation of a new list item.

Edit

The 'Edit' menu item displays the Class Heading Detail dialog to allow for editing the selected list item.

Delete

The 'Delete' menu item deletes the selected list item.

Refresh

The 'Refresh' menu item reloads the list items from the database.

Reset Sequence

The 'Reset Sequence' menu item resequences the items in the lists starting with the number '1'.

Export Text

The 'Export Text' menu item creates a tab delimited text file with the file name 'ExportFiles\ClassHeading.txt'. It attempts to display the data with the application that is associated with the '.txt' extension.

Export CSV

The 'Export CSV' menu item creates a comma delimited text file with the file name 'ExportFiles\ClassHeading.csv'. It attempts to display the data with the application that is associated with the '.csv' extension.

Exit

The 'Exit' menu item closes the window and exits the application.

Help

Displays the context sensitive Help Page.
