Assembly Item Detail Dialog

The Assembly Item Detail dialog allows the user to create and edit assembly item entries.

Form Controls

Assembly Group

The read-only text that shows the parent item description.

Name

The unique item identifier.

Description

The item description text.

Doc File

The compiler generated documentation file path.

Doc File Button

The Doc File Button appears to the right of the Doc File path text. It displays the File Selection dialog for selecting the documentation file.

Image

Path to the image that will display on the Assembly Documentation page.

Image Button

The Image Button appears to the right of the Image path text. It displays the File Selection dialog for selecting the image file.

Sequence

The item sequence.

Active

The CheckBox that indicates this item documentation will be generated.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
