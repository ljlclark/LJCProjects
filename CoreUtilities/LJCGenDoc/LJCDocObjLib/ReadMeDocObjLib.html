LJC DocObjLib Readme

LJCDocDataLib

Transforms the XML comments into the DocData format.

DocData XML is generated from an XML Comment output file.

DataAssembly.cs Represents a DocData Assembly
DataAssemblies.cs Represents a collection of Assembly objects.
DataCommon.cs Contains Common Data Methods
DataExample.cs Represents a DocData Example XML Comment
DataException.cs Represents a DocData Exception XML Comment
DataExceptions.cs Represents a collection of DataException objects.
DataField.cs Represents a DocData Field XML Comment
DataLink.cs Represents a Link
DataLinks.cs Represents a collection of Link objects.
DataMethod.cs Represents a DocData Method XML Comment
DataMethods.cs Represents a collection of Method objects.
DataPara.cs Represents a DocData Paragraph
DataParas.cs Represents a collection of Paragraph objects.
DataParam.cs Represents a DocData Parameter XML Comment
DataParams.cs Represents a collection of Parameter objects.
DataProperty.cs Represents a DocData Property XML Comment
DataProperties.cs Represents a collection of Property objects.
DataRemark.cs Represents a DocData Remark XML Comment
DataRoot.cs Represents the Root object.
DataType.cs Represents a Data Type.
