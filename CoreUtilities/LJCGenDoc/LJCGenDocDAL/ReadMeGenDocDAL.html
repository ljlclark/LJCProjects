LJC GenDocDAL Readme

LJCGenDocDAL

Represents an object model for the code data.

ManagersGenDoc.cs Represents the available GenDoc Manager objects.
ValuesGenDoc.cs Represents the singleton configuration values.

/Data
DocAssembly.cs Represents an Assembly
DocAssemblies.cs Represents a collection of Assembly objects.
DocAssemblyGroup.cs Represents an Assembly Group
DocAssemblyGroups.cs Represents a collection of DocAssemblyGroup objects.
DocClass.cs Represents a Class
DocClasses.cs Represents a collection of DocClass objects.
DocClassGroup.cs Represents a Class Group
DocClassGroups.cs Represents a collection of DocClassGroup objects.
DocClassGroupHeading.cs Represents a Class Group Heading
DocClassGroupHeadings.cs Represents a collection of DocClassGroupHeading objects.
DocMethod.cs Represents a Method
DocMethods.cs Represents a collection of DocMethod objects.
DocMethodGroup.cs Represents a Method Group
DocMethodGroups.cs Represents a collection of DocMethodGroup objects.
DocMethodGroupHeading.cs Represents a Method Group Heading
DocMethodGroupHeadings.cs Represents a collection of DocMethodGroupHeading objects.
