Filter Detail Dialog

The Filter Detail dialog allows the user to create and edit Filter entries.

Form Controls

View Name

The parent View Name. This value is read-only.

Name

The Filter name.

Boolean Operator

The Filter boolean operator.

The Combobox contains the boolean operators "And" and "Or". The operator can be selected from the Combobox or a value can be entered.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
