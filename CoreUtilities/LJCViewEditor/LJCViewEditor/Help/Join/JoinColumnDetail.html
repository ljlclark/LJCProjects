Join Column Detail Dialog

The Join Column Detail dialog allows the user to create and edit Join Column entries.

Form Controls

View Join Name

The parent Join Name. This value is read-only.

Template Column

This Dropdown list contains the columns for the selected Join Table. The related fields: Column Name, Property Name, Caption and Data Type Name are populated with the default values when the Template Column is selected.

Column Name

The Join table column name. This value does not allow blanks.

Property Name

The Join column property name. This value does not allow blanks.

Rename As

The Join column Rename As name. This value does not allow blanks.

Caption

The Join column Caption value.

Data Type Name

The Join column Data Type name value.

This Dropdown list contains the Data Type names from the DataType table.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
