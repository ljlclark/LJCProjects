Column Detail Dialog

The Column Detail dialog allows the user to create and edit Column entries.

Form Controls

View Name

The parent View Name. This value is read-only.

Template Column

This Dropdown List contains the columns for the selected Table. The related fields: Column Name, Property Name, Caption and Data Type Name are populated with the default values when the Template Column is selected.

Column Name

The View table column name. This value does not allow blanks.

Property Name

The View column property name. This value does not allow blanks.

Rename As

The View column Rename As name. This value does not allow blanks.

Caption

The View column Caption value.

Data Type Name

The View column Data Type name value.

This Dropdown list contains the Data Type names from the DataType table.

Value

The View column value.

Primary Key

Indicates if the View column is a primary key.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
