Assignment Statement

An Assignment is used to assign a value to a Data item. It has the following format: 

  DataItemName = Number
  or
  DataItemName = DataItemName/Number Operation DataItemName/Number

The second and fourth positions can be a DataItem name, number or a function with one argument.

The available functions are: cos(), radians(), sin(), sqrt() and tan().

The function names are not case sensitive.

cos(radians) Takes the angle in radians and returns the cosine.
radians(degrees) Takes the angle in degrees and returns the angle in radians.
sin(radians) Takes the angle in radians and returns the sine.
sqrt(4) Takes a number and returns the square root.
tan(radians) Takes the angle in radians and returns the tangent.

This is an example of how to calculate the sides of a 3, 4, 5 triangle. 

  radius = 5
  angle = radians(53.13)
  // cos(angle) = a/h | multiply both sides by h | Cos(angle) * h = a
  // sin(angle) = o/h | multiply both sides by h | Sin(angle) * h = o
  x = cos(angle) * radius
  y = sin(angle) * radius
  value: x round (3)
  value: y round (4)

The Operation can be one of the following: "+", "-", "*", "/" or "^".

The Operation "^" raises the first value to the power of the second value.

The value is assigned to the left side Data Item.

The following assignment sets the value for Family|Families and all related Data Items based on the Conversion Formulas.

Families = 100

The following assignment calclulaes the required water taps at one for each 125 Persons.

Taps = Persons / 125
