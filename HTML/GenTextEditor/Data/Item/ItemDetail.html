The Item Detail Dialog

The Item Detail dialog allows the user to create and edit Item entries.

Form Controls

Section Name

The parent Section name. This value is read-only.

Name

The Item name. This name is not used in the Templates or code generation process. It is only for unique visual identification.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
