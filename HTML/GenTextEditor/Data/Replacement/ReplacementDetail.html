The Replacement Detail Dialog

The Replacement Detail dialog allows the user to create and edit Replacement entries.

Form Controls

Item Name

The parent Item Name. This value is read-only.

Name

The Replacement name. This is the value that appears after the #Value tag in the Template. It will be replaced with the Replacement Value.

Name

The Replacement Value. This is the value that will replace the Replacement Name values in the Template.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
