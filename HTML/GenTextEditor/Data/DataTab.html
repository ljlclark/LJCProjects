The Data Tab

Data Tab Controls

Data TextBox

The Data TextBox contains the name of the XML file retrieved with the Data Button. It is the name of the file where the current data XML will be saved if the "Save" menu item is selected from the Section, Item or Replacement List menus.

The default data XML file path is:
C:\Users\UserName\Documents\Visual Studio 2022\LJCProjectsDev\CoreUtilities\LJCGenText\LJCGenTableCode\bin\Debug\XMLFiles

The target saved XML file name can be changed by editing the name in the Data TextBox.

Data Button

The Data Button displays a File Open dialog to open the desired Data XML file.
