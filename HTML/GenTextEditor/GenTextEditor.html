GenText Editor

The GenText Editor provides for editing Code Templates and Replacement Data. It also provides for generating starter Data items and generating Output Code.

The Feature Tabs

The GenText Editor contains three Feature Tabs:

The Template Tab

The Data Tab

The Output Tab

The Data Item Lists

The Data Item Lists are on the Data Tab.
