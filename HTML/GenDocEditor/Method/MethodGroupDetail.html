The Method Group Detail Dialog

The Method Group Detail dialog allows the user to create and edit Method Group entries.

Form Controls

Class Item Name

The read-only text that shows the Class which contains the methods in this group.

Method Group Name

The unique item identifier.

Class Group Button

The Method Group Button appears to the right of the Method Group Name. It displays the Method Heading Select List for selecting an existing group.

Heading

The item heading text.

When an existing group is selected, this text is read-only and contains the existing group heading.

Custom Heading

The item custom heading text overrides the selected heading text.

Sequence

The item sequence.

Active

The CheckBox that indicates this item documentation will be generated.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
