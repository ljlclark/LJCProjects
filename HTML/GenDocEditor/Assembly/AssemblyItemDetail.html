The Assembly Item Detail Dialog

The Assembly Item Detail dialog allows the user to create and edit assembly item entries.

Form Controls

Assembly Group

The read-only text that shows the parent item description.

Name

The unique item identifier.

Description

The item description text.

Doc File

The compiler generated documentation file path.

Doc File Button

The Doc File Button appears to the right of the Doc File path text. It displays the File Selection dialog for selecting the documentation file.

Image

Path to the image that will display on the Assembly Documentation page.

Image Button

The Image Button appears to the right of the Image path text. It displays the File Selection dialog for selecting the image file.

Sequence

The item sequence.

Active

The CheckBox that indicates this item documentation will be generated.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Save and New

Saves the current item and clears the detail dialog ready for a new item.

Save and Previous

Saves the current item and loads the detail dialog with the data for the previous item in the parent list ready for editing.

The dialog is dismissed if the current item was the first item in the list.

Save and Next

Saves the current item and loads the detail dialog with the data for the next item in the parent list ready for editing.

The dialog is dismissed if the current item was the last item in the list.

Help

Displays the context sensitive Help Page.
