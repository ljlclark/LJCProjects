Assembly Tab

The GenDoc Editor contains the Assembly Tab which has the Assembly Group and Assembly Item lists.

The Assembly Group list contains the defined groups for display on the Main HTML documentation page.

The Assembly Item list contains the defined assemblies for the selected Assembly Group.
