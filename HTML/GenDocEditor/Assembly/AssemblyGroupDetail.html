Assembly Group Detail Dialog

The Assembly Group Detail dialog allows the user to create and edit Assembly Group entries.

Form Controls

Assembly Group Name

The unique item identifier.

Heading

The item heading text.

Sequence

The item sequence.

Active

The CheckBox that indicates this item documentation will be generated.

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.
