How to Add LJC Custom Controls to the Toolbox

View the Designer for a form in your project.

Select the ToolBox tab. If the ToolBox tab is not shown, select the "View" menu and then "ToolBox".

Click the right mouse button anywhere in the ToolBox and select the "Add Tab" menu item. Name the tab "LJCControls".

Click the right mouse button on the "LJControls"" tab and select the "Choose Items" menu item.

Click the "Browse" button and navigate to "CoreAssemblies\LJCLibraries\LJCWinFormControls\bin\Debug.

Select the file LJCWinFormControls.dll.
