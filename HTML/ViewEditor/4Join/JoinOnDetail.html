JoinOn Detail

The JoinOn Detail dialog allows the user to create and edit JoinOn entries.

Form Controls

Join Name

The parent Join Name. This value is read-only.

From Column Name

The Join from column name. This value does not allow blanks.

The Combobox contains the column names from the View Table. The column name can be selected from the Combobox or a name can be entered.

To Column Name

The Join to column name. This value does not allow blanks.

The Combobox contains the column names from the Join Table. The column name can be selected from the Combobox or a name can be entered.

Join On Operator

The JoinOn operator. This value does not allow blanks.

The user can enter any valid join operator. The most commonly used value is "=".

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on the Combobox control.

Help

Displays the context sensitive Help Page.
