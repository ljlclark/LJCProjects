Condition Detail

The Condition Detail dialog allows the user to create and edit Condition entries.

Form Controls

Condition Set Name

The parent ConditionSet Name. This value is read-only.

First Value

The compare column name Dropdown. This value does not allow blanks.

The Combobox contains the column names from the View Table. The column name can be selected from the Combobox or a name can be entered.

Second Value

The compare-to column name or value. This value does not allow blanks.

This value would generally be a column name from the Join Table. One could however use a constant or other valid value.

Comparison Operator

The comparison operator. This value does not allow blanks.

The user can enter any valid comparison operator. The most commonly used value is "=". Some other possible values include: ">" or "<".

OK Button

Saves the entered data and exits the dialog.

Cancel Button

Exits the dialog without saving the entered data.

Context Menu

The Context Menu can be displayed by clicking the right mouse button on a place in the dialog that does not contain a control item.

Help

Displays the context sensitive Help Page.
