The Action Methods

Each list grid has action methods which are triggered from a tool icon selection or menu item selection.
These methods are DoNew(), DoEdit(), DoDelete(), DoRefresh() and DoSelect().
The methods may be postfixed with a value that identifies the grid to which it applies. E.G. DoNewAddress(), DoEditAddress(), etc.
This is especially true if the form contains more than one grid.
